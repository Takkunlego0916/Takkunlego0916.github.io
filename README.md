# Game Portal

GitHub Pages 上で遊べる ブラウザゲーム集 です。
すべて HTML / CSS / JavaScript だけで作られており、インストール不要でどのデバイスからでも遊べます。

# 公開ページ

ゲームホームページ（ポータル）
[プレイ](https://takkunmc.sytes.net/)

# 収録ゲーム一覧
**1️⃣2048**

同じ数字のタイルを合体させて 2048 を作る定番パズル。
矢印キー or スワイプで操作。

[プレイ](https://takkunmc.sytes.net/2048/)

主な特徴
	•	スマホ & PC 対応
	•	スワイプ操作
	•	2048 達成演出
	•	シンプルで高速

**2️⃣ Tetris（テトリス）**

落ちてくるテトロミノを回転・移動させてラインを消すパズルゲーム。

[プレイ](https://takkunmc.sytes.net/tetris/)

主な特徴
	•	スムーズな落下アニメーション
	•	ライン消去
	•	Game Over 画面つき

**3️⃣ Othello（オセロ / リバーシ）**

黒と白の石を交互に打ち、挟んだ石をひっくり返すボードゲーム。
（シンプルな AI や PvP モードあり ※仕様に合わせて調整）

[プレイ](https://takkunmc.sytes.net/Othello/)
特徴
	•	操作しやすい 8×8 盤面
	•	合法手のハイライト
	•	石数カウント
	•	ゲーム終了判定あり
